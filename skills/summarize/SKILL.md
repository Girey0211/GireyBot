# 대화 요약 스킬

## 트리거 조건

- "요약해줘", "대화 정리해줘", "무슨 얘기 했어?" 등 대화 요약 요청
- `/skill summarize` 슬래시 명령어

## 컨텍스트

이 스킬은 `context_mode: session`으로 동작합니다.
컨텍스트에는 현재 채널의 **대화 세션** 전체가 포함됩니다.
대화 세션은 가장 최근 메시지부터 역순으로 탐색하여,
연속된 두 메시지 사이에 10분 이상 공백이 있는 지점에서 끊깁니다.

## 요약 규칙

1. 컨텍스트로 전달된 대화 세션 전체를 분석한다.
2. 다음 형식으로 요약을 생성한다:

### 요약 형식

- **한 줄 요약**: 전체 대화의 핵심을 한 문장으로 압축
- **주요 내용**: 논의된 주제별로 2~5개 항목으로 정리
- **참여자**: 대화에 참여한 사람 목록
- **결론/미결**: 합의된 사항이나 아직 결정되지 않은 사항

### 요약 원칙

- 사실 기반으로 작성한다. 추측이나 해석을 넣지 않는다.
- 중요한 결정, 약속, 할 일은 반드시 포함한다.
- 감정적 뉘앙스는 생략하고 내용에 집중한다.
- 대화가 없거나 너무 짧으면 "요약할 대화가 부족합니다"라고 안내한다.

## 출력 형식

마크다운으로 응답한다 (Embed 안에 표시됨):

```
**한 줄 요약**: ...

**주요 내용**:
- ...
- ...

**참여자**: 이름1, 이름2

**결론/미결**:
- ✅ ...
- ❓ ...
```

## 에러 처리

- 대화 기록 없음 → "이 채널에 최근 대화 기록이 없습니다."
- 메시지 1~2개 → "요약할 만큼 충분한 대화가 없습니다. (N개 메시지)"
